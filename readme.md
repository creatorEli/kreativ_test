# Тренажёр - тестовое задание

Репозиторий для тестового задания для отбора на вакансию в Креативное дело

Включает в себя:

+ __index.html__ - файл с вёрсткой тренажёра
+ __results.html__ - файл со свёрстанной таблицей результатов

Также реализован функционал тренажёра на JavaScript.

Тренажёр состоит из пяти заданий. По результатам прохождения формируется таблица в results.html со всеми резульатами
